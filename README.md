## 核心架构

### 贝叶斯渐进式画像引擎
每一句话都是一次贝叶斯更新。系统从对话中提取无意识行为模式——用词选择、句式风格、语气词频率——这些最难伪装的信号，构建动态人格画像。置信度诚实反映证据充分程度，永远不会到 100%。

### 五层信号提取框架
从浅到深，逐层解锁：
1. **表层语言特征** — 即使 1 句话也能开始提取（用词正式程度、句长、标点习惯、表情偏好）
2. **话语结构特征** — 2-3 句话后可提取（先说结论还是铺垫、反问频率、转折词使用）
3. **互动模式特征** — 3-5 轮对话后可提取（回复速度、主动提问、冲突处理模式）
4. **内容语义特征** — 5-10 轮对话后可提取（核心主题、归因风格、时间表达、价值信号）
5. **元认知特征** — 低频但高价值信号（自我修正、不确定性表达、情绪自我觉察）

### 双视角画像系统
每个人物画像由两部分构成：
- **AI 分析视角** — 基于对话数据的客观行为建模（8维人格 + 贝叶斯置信度）
- **用户主观视角** — 你对此人的个人印象、关系描述、信任度、未解决问题

### 证据链 & 画像版本控制
每个画像判断都可追溯到具体的对话原文。画像随着新证据自动演进，历史版本完整保留。

## 功能模块矩阵

SIOS 包含 **12 大核心模块 + 多项附加功能**，覆盖从数据分析到心灵探索的全维度：

### 📊 1. 数据大盘 (Analytics Dashboard)
一站式全局总览——对话统计、画像分布、成就进度、活跃趋势、AI 洞察报告。
- 多维数据可视化（Recharts）
- 一键 AI 数据洞察（自动识别趋势和异常）
- 分享旅程卡片

### 💬 2. 对话分析 (Conversation Analysis)
粘贴对话 → 五层深度分析 → 情绪曲线 + 关键转折点 + 策略洞察 + MBTI推测 + 画像自动生成。
- 支持结构化对话和无标记对话（微信直接粘贴）
- 智能归因编辑器：自动猜测说话人，支持手动修正
- 群聊分析：多人对话 → 群体动力学 + 社交网络图谱
- 对话原文持久化存储 + 历史浏览 + 重新分析
- 内置 9 大类 30+ 示例对话

### 👥 3. 人物画像 (Person Profiles)
8 维人格雷达图 + 置信度 + 沟通风格 + 行为模式 + 情绪触发点 + 说服策略。
- **双视角画像**：AI分析 + 用户主观印象
- **自我画像**：跨对话沟通模式、语言指纹、盲点与成长建议
- **关系温度趋势**：情感轨迹可视化 + 自动升温/降温判定
- **智能关系维护提醒** + 画像去重合并
- 关系网络图谱（D3.js力导向图）
- 跨模块一键导航

### ⚔️ 4. 模拟演练 (Simulation & Coaching)
集实时教练、模拟对练、情商训练、策略规划于一体的综合训练中心。
- **实时教练**：博弈论思维 + 话术建议 + 风险预警
- **模拟对练**：与 AI 数字分身进行三级难度对话演练
- **情商训练**：5维能力评分 + 逐句复盘 + 画像融合
- **策略规划**：BATNA + 心理策略 + 完整对话准备
- 剧本生成器：场景化沟通剧本
- 120+ 内置训练场景

### 🧠 5. 心理顾问 (Psychology Counselor)
基于关系网络和画像数据，提供个性化心理疏导与关系优化建议。
- 六步分析框架：情绪识别 → 需求解码 → 模式识别 → 认知审视 → 资源盘点 → 行动建议
- 自动整合画像数据，精准人际关系洞察
- 多轮对话式咨询 + 语音输入
- 专业边界提醒

### ⚖️ 6. 法律顾问 (Legal Advisor)
AI 法律咨询助手，提供法律分析、合同审查、权益保护建议。
- 多轮法律问题咨询
- 图片识别（合同/文件 OCR）
- 免责声明 + 专业律师建议

### 📅 7. 规划制定 (Life & Work Planner)
输入目标 → AI 生成多阶段行动计划，包含里程碑、时间线、资源需求、风险预警。
- 7 大领域 × 5 种时间尺度
- 高级选项：预算约束、偏好设置、自定义约束
- 内置 6 大类 15+ 规划模板

### 🔮 8. 风水玄学 (Metaphysics & Divination)
集成 11 种传统文化测算体系的专业互动模块。
- **周易占卜**（梅花易数 · 六十四卦） · **八字命理**（四柱干支 · 十神格局）
- **风水布局**（玄空飞星 · 八宅方位） · **紫微斗数**（十四主星 · 十二宫位）
- **塔罗牌阵**（韦特体系 · 多种牌阵） · **奇门遁甲**（三奇六仪 · 八门九星）
- **六爻预测** · **姓名学** · **求签解签** · **测字解字** · **面相手相** · **解梦析梦**
- 交互式仪式（铜钱摇卦、塔罗抽牌、求签动画等）
- 图片上传（面相/手相识别）

### 🧬 9. MBTI 人格测试 (Personality Quick Test)
三种深度模式，93题精准测评 + 16型详细报告。
- **快速模式**（24题 / 3分钟）· **标准模式**（48题 / 8分钟）· **完整模式**（93题 / 15分钟）
- 认知功能栈、沟通风格、冲突处理、压力反应、成长建议
- 结果全系统联动

### ⚙️ 10. 数据管理 (Data & Settings)
- 数据导出/导入（JSON格式）
- API 配置面板（多模型切换：MiniMax / Claude / GPT-4o / DeepSeek）
- 隐私设置 + 成就系统（30+ 成就）
- 主题切换 + 中英双语

- **智能消息归因**：AI 自动猜测说话人 + 可视化编辑器

## 快速开始

### 方式一：在线体验（推荐）

直接访问 👉 [**http://39.108.112.161/**](http://39.108.112.161/)

### 方式二：本地开发

```bash
# 克隆项目
git clone https://github.com/JingSir-Git/SIOS.git
cd SIOS

# 安装依赖
npm install

# 配置环境变量
cp env.example .env.local
# 编辑 .env.local，填入你的 API 密钥

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 即可使用。

> 💡 也可以不配置 .env.local，启动后在「设置 → API 配置」中直接填写 API Key。

### 方式三：Windows 桌面版

1. 从 [Releases](https://github.com/JingSir-Git/SIOS/releases) 下载最新 `.exe` 安装包
2. 双击安装，桌面快捷方式启动
3. 首次启动在「设置 → API 配置」中填写你的 API Key
4. 所有数据保存在本地，无需联网（仅 AI 对话需要网络）

### 方式四：服务器部署

```bash
# 一键部署到远程服务器（本地构建 + 上传 + PM2 重启）
bash deploy.sh
```

## 隐私架构

隐私是产品的核心信任基础，从第一天起就按最高标准设计：

- **本地优先**: 所有画像和对话数据存储在用户本地（浏览器 localStorage / 桌面版本地文件）
- **按需调用**: API 调用仅在用户主动操作时发生，不做后台数据同步
- **零收集**: 不收集任何用户行为数据、不追踪、不上报
- **完全控制**: 所有数据可随时查看、导出（JSON）、删除
- **桌面版完全离线**: Electron 桌面版仅在 AI 对话时需要网络

## 设计哲学

> **情商的本质是理解他人并建立真实连接，不是操纵。**

系统始终引导用户寻找双赢方案，而非单方面利用对方弱点。
AI 的输出从第一天就传递正确的价值观——帮你成为更好的沟通者，而不是更好的操控者。
